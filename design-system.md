# VexarLabs Design System - Creative & Bold

## Design Philosophy
**Creative/Bold Tech Studio** - Breaking away from conventional tech aesthetics with vibrant colors, asymmetric layouts, and dynamic visual elements.

## Color Palette
### Primary Colors
- **Coral Orange**: #FF6B4A (primary actions, highlights)
- **Deep Orange**: #E85D3C (hover states)
- **Sunset Orange**: #FF8C6B (gradients, accents)

### Secondary Colors
- **Cyan**: #06B6D4 (secondary actions, tech elements)
- **Turquoise**: #14B8A6 (accents, highlights)
- **Teal**: #0D9488 (hover states)

### Neutral Colors
- **Charcoal**: #1F2937 (text, dark elements)
- **Slate**: #475569 (secondary text)
- **Light Gray**: #F1F5F9 (backgrounds)
- **White**: #FFFFFF (cards, sections)

## Typography
- **Headings**: Outfit (bold, modern)
- **Body**: DM Sans (clean, readable)

## Layout Principles
1. **Asymmetric sections** - Alternating left/right content blocks
2. **Diagonal cuts** - Using clip-path for dynamic section transitions
3. **Overlapping elements** - Cards and images breaking container bounds
4. **Generous whitespace** - Breathing room between sections
5. **Bold typography** - Large, impactful headings

## Visual Elements
- **Gradient overlays** - Orange to cyan gradients
- **Geometric patterns** - Subtle background patterns
- **Shadow depth** - Layered shadows for depth
- **Rounded corners** - Consistent 12px-24px radius
- **Animated hover states** - Smooth transitions and scale effects

## Component Styling
- **Cards**: White background, subtle shadow, hover lift effect
- **Buttons**: Bold coral orange, white text, rounded
- **Icons**: Cyan/turquoise for tech elements
- **Images**: Rounded corners, subtle border
